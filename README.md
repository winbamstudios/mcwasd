# mcwasd
markov chain with a sunny disposition<br>
a basic n-gram language model!!! to use it, just compile it. syntax is
```
mcwasd [training data] [temperature, 0 through 10]
temperature of 3 is used if no value is input
```
<br>training data can be any plain text file. yes, anything. as long as your computer is decently fast the training shouldn't take long
